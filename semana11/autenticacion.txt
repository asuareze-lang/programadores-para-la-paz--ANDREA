El login es el proceso mediante el cual un usuario se identifica en una aplicación usando sus credenciales, como un correo electrónico y una contraseña. Esto permite verificar su identidad antes de acceder a determinadas funciones. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes que no deben estar disponibles para cualquier persona. Además, no se deben exponer públicamente contraseñas, documentos de identidad, teléfonos, direcciones, tokens, claves de API ni otros datos sensibles de la comunidad.
