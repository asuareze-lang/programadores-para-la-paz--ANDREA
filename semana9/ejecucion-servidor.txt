Al ejecutar el comando node server.js, Node.js inicia la aplicación y pone en funcionamiento el servidor.

En la terminal aparece el mensaje: "Servidor ejecutándose en puerto 3000".

Cuando visito la ruta /estado en el navegador, el servidor responde con un objeto JSON que indica que el servidor está funcionando correctamente y que el servicio corresponde a una API comunitaria.

Respuesta obtenida:

{
"estado": "Servidor funcionando",
"servicio": "API comunitaria"
}
