Sistema operativo:
Es el software principal del computador o servidor. Administra los recursos del hardware y permite ejecutar aplicaciones.

Node.js:
Es un entorno de ejecución que permite correr código JavaScript fuera del navegador. Se utiliza para crear servidores y aplicaciones web.

Aplicación server.js:
Es el archivo que contiene el código del servidor. Define las rutas y respuestas que recibirán los usuarios.

Puerto:
Es un canal de comunicación utilizado por las aplicaciones para recibir solicitudes. En esta práctica se utiliza el puerto 3000.

Logs:
Son registros que almacenan eventos, actividades y errores del sistema. Ayudan a supervisar y solucionar problemas en un servidor.
