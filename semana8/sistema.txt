El comando uname -a muestra información del sistema operativo, incluyendo el nombre del sistema, versión del kernel y arquitectura del equipo.

El comando whoami muestra el nombre del usuario que está utilizando el sistema en ese momento.

Conocer esta información es útil al administrar un servidor porque permite identificar el sistema operativo instalado, verificar la cuenta con la que se está trabajando y facilitar tareas de configuración, mantenimiento y seguridad.
